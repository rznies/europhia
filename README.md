# europhia

A food and beverage landing page featuring dynamic menus and event tracking powered by Genkit and Express.

Built using **Software Development Workspace**.

### Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/rznies/europhia.git
   ```
2. Install dependencies (if applicable):
   ```bash
   npm install
   ```
3. Run the development environment or execute scripts according to project files.
